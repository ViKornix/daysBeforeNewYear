# Days Before New Year API

Простой HTTP API на NestJS, который возвращает:
- сколько дней осталось до Нового года;
- текущее локальное время сервера.

## Стек

- Node.js
- TypeScript
- NestJS 11

## Требования

- Node.js 18+ 
- npm

## Быстрый старт

```bash
npm install
npm run start:dev
```

После запуска сервис доступен по адресу:

`http://localhost:3000`

## API

### `GET /`

Возвращает JSON с количеством дней до 31 декабря текущего года и текущим временем.

Пример ответа:

```json
{
  "data": {
    "daysBeforeNewYear": 307,
    "now": "2/28/2026, 2:20:00 PM"
  }
}
```

## Скрипты

- `npm run start` - запуск приложения
- `npm run start:dev` - запуск в режиме разработки с watch
- `npm run build` - сборка в `dist/`
- `npm run start:prod` - запуск собранного приложения

## Замечания

- Значение `daysBeforeNewYear` считается на момент запроса.
- Время `now` зависит от локали и часового пояса окружения, где запущен сервер.
